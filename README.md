# FourierAnalysisForSignalReconstruction
This project is a basic exercise and introduction to computational modelling. The code generates a noisy multi-frequency signal which is plotted on a time-domain and then converted to a frequency domain through the fast Fourier transform algorithm. The peak frequencies are then identified and the original signal is reconstructed.
